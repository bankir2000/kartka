🗺 Картка вогню

📖 Інструкція з використання

🎯 Що робить програма

  • 📍 Будує азимутну сітку 1-4 км на топографічній карті
  • 🟢 Позначає орієнтири з відстанню в гектометрах
  • 🔴 Позначає заборонені сектори обстрілу
  • 📄 Генерує PDF документ «Картка вогню»

🔒 Кібербезпека

  • ✅ Координати позиції — нікуди не передаються
  • ✅ PDF генерується локально на пристрої
  • ✅ Реквізити командира — зберігаються тільки на пристрої
  • ⚠️ Тайли карти завантажуються з інтернету — сервер бачить район роботи

📥 Безпечна робота офлайн

1. Кешуйте область заздалегідь

Натисніть «📥 Кешувати область» → оберіть потрібну область → zoom 10, 11, 12 → «Почати завантаження». Зачекайте завершення (5-15 хвилин).

2. Вмикайте режим «В літаку»

Після кешування завжди працюйте з вимкненим інтернетом. Карта, сітка та PDF працюють повністю офлайн.

3. Очищайте дані після роботи

Chrome → ⋮ → Налаштування → Конфіденційність → Видалити дані перегляду → Кешовані зображення. Або використовуйте режим інкогніто.

📋 Як користуватися

Крок 1 — Реквізити (необов'язково)

Натисніть «⚙️ Налаштування реквізитів» → введіть звання та ПІБ командира → збережіть. Ці дані автоматично потраплять у PDF. Якщо не заповнити — у PDF залишаться порожні рядки для підпису вручну.

Крок 2 — Координати

Введіть координати центру позиції. Підтримується формат з крапкою (50.123456, 36.123456) та з комою (50,123456, 36,123456).

Крок 3 — Орієнтири (за потреби)

Введіть координати орієнтирів. Програма автоматично розрахує відстань від позиції до кожного орієнтиру.

Крок 4 — Червоні зони (за потреби)

Введіть діапазон градусів забороненого сектору, наприклад: 0-30. Можна додати кілька зон.

Крок 5 — Побудова і PDF

Натисніть «🏹 ПОБУДУВАТИ АЗИМУТНУ СІТКУ» → перевірте результат на карті → натисніть «📄 ЗБЕРЕГТИ PDF».

⚡ Швидкі поради

📲 Встановлення як окремої програми

Програма встановлюється на телефон як окремий додаток — без браузера, з іконкою на головному екрані.

Android Chrome: ⋮ → «Додати на головний екран» → «Встановити»

iOS Safari: кнопка «Поділитися» → «На екран Home»

Після встановлення програма працює як нативний додаток — окреме вікно, без адресного рядка браузера, повністю офлайн після кешування.

  • 🔄 Координати з комою замість крапки — виправляються автоматично
  • 📄 Назва PDF файлу = назва позиції (поле «Назва позиції»)
  • 💾 Реквізити зберігаються між сесіями автоматично
